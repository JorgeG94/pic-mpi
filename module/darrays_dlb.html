Dynamic load balancing for distributed arrays

Provides atomic counter-based load balancing using MPI-3 RMA. Rank 0 hosts the counter, all ranks can atomically increment it.


Uses

    • pic_mpi_lib
    • mpi_f08
    • pic_types
